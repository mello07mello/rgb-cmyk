Diferença entre CMYK e RGB

O modelo CMYK é utilizado para impressão, enquanto o modelo RGB é usado para telas digitais.

RGB

Usado para telas digitais.

Componentes: Vermelho, Verde e Azul.

Exemplo de cor: RGB(255, 0, 0) → Vermelho

CMYK

Usado para impressão.

Componentes: Ciano, Magenta, Amarelo e Preto.

Exemplo de cor: Magenta.
