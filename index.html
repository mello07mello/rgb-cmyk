Entendendo os Modelos de Cores: RGB e CMYK

Uma explicação detalhada sobre os sistemas de cores aditivo e subtrativo

Modelo RGB (Aditivo)

O modelo RGB é um sistema aditivo de cores usado em dispositivos eletrônicos como monitores, smartphones e TVs. Neste modelo, as cores são criadas pela adição de luz nas três cores primárias:

  • R - Vermelho (Red)
  • G - Verde (Green)
  • B - Azul (Blue)

Estrutura de um pixel RGB

Cada cor tem uma intensidade que varia de 0 a 255. Por exemplo:

  • RGB(255, 0, 0) = Vermelho puro
  • RGB(255, 255, 0) = Amarelo
  • RGB(255, 255, 255) = Branco

Modelo CMYK (Subtrativo)

O CMYK é um modelo subtrativo usado principalmente em impressão. As cores são formadas pela subtração de luz através de pigmentos:

  • C - Ciano (Cyan)
  • M - Magenta (Magenta)
  • Y - Amarelo (Yellow)
  • K - Preto (Key/Black)

O preto (K) é adicionado por razões práticas:

  • Economia de tinta colorida
  • Melhor contraste e profundidade
  • Textos mais nítidos

Comparação entre RGB e CMYK

RGB (Aditivo)

  • Usado em dispositivos eletrônicos
  • Adiciona luz para criar cores
  • Ausência de luz = Preto
  • Todas as cores = Branco

CMYK (Subtrativo)

  • Usado em impressão
  • Subtrai luz para criar cores
  • Ausência de tinta = Branco
  • Todas as cores = Preto

Como os Pixels Funcionam

Cada pixel em uma tela é composto por três subpixels (vermelho, verde e azul). A intensidade de cada subpixel determina a cor final que vemos. Quando todos os subpixels estão em intensidade máxima, vemos branco. Quando todos estão desligados, vemos preto.
